News Archive:
10/2/99 to 3/29/2K

3/29/2K - All righty then. Its been a while since my last update. I've been spending this time creating a new Quake2 PPM, one which I first modelled for a contest at Polycount. The model is finished, and now makes its home on Quake2 / PPMs. Speaking of the PPMs page, the Future Models section has been removed, because I'm going to be bogged down with the mind numbing work I have to do in order to get the first few chapters of Into the Fray: Hour of Need finished by the end of the year. You can find out more about the progress I make on this daunting project as I get it done by checking out the soon to be opened Into the Fray / Internet Show page. Getting this project done is sure to be a bitch, but then again, so is life.

2/11/2K - Check out the all-new House of Cantar page in the Into the Fray / Tour Guide section of IWOW. All right, I know it's been ages since I updated the Quake2 section, but here goes. 12/25/99 - Merry Christmas, visitors! The races page is up and running, and it already comes with two races, quinnisalzes and ambrons. Also, the Chiddetny Page under Into the Fray / Tour Guide has been updated.
