My Recorded Games

> These are some of the games I played over QuakeCon's network. These are not standard movie files; in order to view them, you have to run them via the games they were recorded in. In case you don't already know how to view recored games, I've included instructions on how to do this for each game.

Quake 2 - Make sure you have the latest version of Quake 2 (v3.20). Copy the recorded game to your Quake2/baseq2/demos directory. If there is no demos directory under baseq2, create one. Run Quake 2, and in the console, type in "map {filename}", where {filename} is the name of the recorded game, including the file extention which is usually dm2.

Generations Arena - Make sure you have the latest version of Quake 3 Arena, and the latest version of Generations. Copy the recorded game to your Quake III Arena/generations/demos directory. If there is no demos directory under generations, create one. Run the Generations mod, click on Demos in the main menu,  select the demo you downloaded, and then click Play Demo.

Starcraft - Make sure you have the latest version of Starcraft: Brood War. Copy the recorded game to your Starcraft/maps/replays directory. Run Starcraft, and in the main menu, select Single Player, and then Starcraft: Brood War Expansion. In the registry screen, sign on using any profile, but create a new profile if the list is blank. Click the Load Replay button, and in the list of replays, select the file you downloaded, then click OK.
